
Smart Store - Учебный статический сайт
--------------------------------------
Файлы:
- index.html - главная страница с каталогом
- checkout.html - оформление заказа (демо)
- styles.css - стили
- app.js - логика каталога и модального окна
- checkout.js - логика корзины и оформления
- images/ - svg-изображения-заполнители

Как запустить локально:
1. Распакуйте проект.
2. Откройте index.html в браузере или используйте Live Server в VS Code.
3. Чтобы загрузить в GitHub - создайте репозиторий и скопируйте файлы, затем сделайте git push.

Примечание: это учебный проект, оплата не выполняется - демонстрация поведения корзины и оформления.
